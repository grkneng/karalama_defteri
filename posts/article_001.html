JavaScript ES2024 Yenilikler: Derinlemesine İnceleme

Ahmet Yılmaz • 2 Kasım 2025 • 12 dakika okuma

Şekil 1: Docker Konteyner Mimarisi ve Katmanlı Dosya Sistemi

1. Giriş

JavaScript, web geliştirmenin kalbi olmaya devam ediyor. ES2024 (ECMAScript 15), dile yeni güçlü özellikler ekleyerek geliştirici deneyimini önemli ölçüde iyileştiriyor. Bu makalede, bu yeni özellikleri detaylı bir şekilde inceleyeceğiz.

Modern JavaScript uygulamaları yazarken, bu yeni API'ları kullanarak kodunuzu daha temiz, daha performant ve daha bakımlı hale getirebilirsiniz.

2. Matematiksel Temeller

Promise ve asynchronous işlemleri anlamak için bazı matematiksel kavramları incelemek faydalı olacaktır.

Promise Çözüm Formülü

Bir Promise'in çözüm süresi aşağıdaki formülle hesaplanabilir:

$$T_{resolve} = \int_0^{\infty} e^{-\lambda t} \, dt = \frac{1}{\lambda}$$

Performans Hesaplaması

Array işlemlerinin zaman karmaşıklığı:

$$O(n) = \sum_{i=1}^{n} \text{operasyon}_i$$

Paralel İşleme

Workers kullanarak paralel işleme gücü:

$$P = W \times (1 - s + \frac{s}{W})^{-1}$$

Amdahl Yasası: W = worker sayısı, s = seri kod oranı

3. Kod Örnekleri

Promise.withResolvers()

ES2024'ün en faydalı eklentilerinden biri Promise.withResolvers():

// Eski yol
const promise = new Promise((resolve, reject) => {
    // resolve ve reject'i başka yerlerde kullan
});

// Yeni yol - çok daha temiz!
const { promise, resolve, reject } = Promise.withResolvers();

// Şimdi resolve ve reject'i doğrudan kullanabilirsin
setTimeout(() => resolve('Başarılı!'), 1000);

Array Yöntemleri

Array metotlarında önemli gelişmeler:

const numbers = [1, 2, 3, 4, 5];

// toReversed() - orijinal array'i değiştirmez
const reversed = numbers.toReversed();
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(reversed); // [5, 4, 3, 2, 1]

// toSorted() - orijinal array'i değiştirmez
const sorted = numbers.toSorted((a, b) => b - a);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(sorted); // [5, 4, 3, 2, 1]

// with() - immutable güncelleştirme
const updated = numbers.with(2, 99);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(updated); // [1, 2, 99, 4, 5]

String Padding Metotları

// String işleme ile ilgili yeni metotlar
const str = "Hello";

// Başa padding ekle
console.log(str.padStart(10, "*")); // *****Hello

// Sona padding ekle
console.log(str.padEnd(10, "-")); // Hello-----

// Regular expression lookahead
const regex = /\d+(?=px)/;
console.log("100px".match(regex)); // ["100"]

Regular Expression Enhancements

// hasIndices flag ile capturing group konumlarını al
const regex = /(\d+)-(\d+)/d;
const match = "2025-11-02".match(regex);

console.log(match.indices); // [[0, 10], [0, 4], [5, 10]]
console.log(match.indices[1]); // [0, 4] -> "2025"

// Named capture groups
const dateRegex = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/;
const dateMatch = "2025-11-02".match(dateRegex);
console.log(dateMatch.groups); 
// { year: '2025', month: '11', day: '02' }

4. İleri Konular

WeakMap ve WeakSet Optimizasyonları

Bellek yönetimi ES2024'te daha verimli hale geldi:

// WeakMap ile yalnızca referansta tutulan veriler
const cache = new WeakMap();

class User {
    constructor(name) {
        this.name = name;
    }
}

const user = new User("Ahmet");
cache.set(user, { lastLogin: Date.now() });

// User garbage collected olduğunda,
// WeakMap'teki veri de otomatik temizlenir

// Object.entries() ile tüm girişleri getir
const obj = { a: 1, b: 2, c: 3 };
for (const [key, value] of Object.entries(obj)) {
    console.log(`${key}: ${value}`);
}

Concurrent Features

ES2024'te concurrent programlama daha güvenli hale geldi:

// Shared Array Buffer güvenliği
const sharedBuffer = new SharedArrayBuffer(4 * 4);
const sharedArray = new Int32Array(sharedBuffer);

// Atomic işlemler thread-safe
Atomics.store(sharedArray, 0, 42);
console.log(Atomics.load(sharedArray, 0)); // 42

// Multi-threaded çalışma
const worker = new Worker('worker.js');
worker.postMessage({ type: 'compute', data: sharedBuffer });

worker.onmessage = (event) => {
    console.log('İşlem tamamlandı:', event.data);
};

Generator ve Iterator Enhancements

// Async generator ile stream işleme
async function* fetchData(urls) {
    for (const url of urls) {
        const response = await fetch(url);
        yield await response.json();
    }
}

// Kullanım
for await (const data of fetchData(urlArray)) {
    console.log('Veri alındı:', data);
}

5. Sonuç

ES2024, JavaScript'i daha güçlü, daha güvenli ve daha ergonomik hale getiriyor. Bu yeni özellikler, especially Promise.withResolvers(), Array immutability metotları ve improved regular expressions, günlük JavaScript geliştirmede çok faydalı olacaktır.

Temel Çıkarımlar:
  • Promise.withResolvers() daha temiz async kod yazmanızı sağlar
  • Immutable array metotları functional programming'i kolaylaştırır
  • Geliştirilmiş regex özellikleri text işlemeyi güçlendirir
  • Concurrent primitives multi-threaded uygulamalar için safer'dir

Bu özelliklerle çalışma pratiği yaparak, modern JavaScript'te gerçekten ustalık kazanabilirsiniz!
