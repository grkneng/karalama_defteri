JavaScript Event Loop Derinlemesine İnceleme

Author ID · 15 Ocak 2026

JavaScript tek iş parçacıklı bir çalışma modeline sahiptir. Event Loop; Call Stack, Callback Queue ve Microtask Queue arasında koordinasyon sağlar.

Call Stack

Stack yapısı LIFO prensibiyle çalışır.

Matematiksel Model

Kuyruk gecikmesi yaklaşık olarak: $$ L = \lambda W $$ burada \( \lambda \) giriş oranı, \( W \) ise ortalama bekleme süresidir.

Kod Örneği


console.log("Start");

setTimeout(() => {
  console.log("Timeout");
}, 0);

Promise.resolve().then(() => {
  console.log("Promise");
});

console.log("End");

Microtask Önceliği Sonuç

Promise callback'leri, macro task'lardan önce çalıştırılır.
