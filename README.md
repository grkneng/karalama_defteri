# ✍️ Karalama Defteri

Bu depo, teknik bloglar, dokümantasyon sayfaları ve kişisel arşivler için tasarlanmış modern, hızlı ve duyarlı bir web şablonudur.

**Amacı:** Hızlı yüklenen, mobil uyumlu ve zengin içerik formatlarını (Markdown, Kod, Matematiksel Formüller) 
destekleyen bir platform sunmaktır.

## ✨ Özellikler

* **Duyarlı Tasarım (Responsive):** Bootstrap 5.x ile mobil cihazlara tam uyum.
* **Lazy Loading (Tembel Yükleme):** Ana sayfada (feed) gönderiler, performans artışı için parça parça yüklenir.
* **Modüler Bileşenler:** Navbar ve Footer, bakım kolaylığı için ayrı HTML dosyalarına ayrılmıştır.
* **Markdown Desteği:** `md.html` sayfası, Markdown dosyalarını (Marked.js) anında HTML'e çevirir.
* **Kod Vurgulama:** Highlight.js kütüphanesi ile tüm kod blokları şık ve okunur bir şekilde renklendirilir.
* **Matematiksel Formül Desteği:** KaTeX kütüphanesi ile LaTeX formatındaki formüller doğru şekilde render edilir.
* **Dinamik Menü:** Dokümantasyon (`md.html`) menüsü, harici bir `md.json` dosyasından kategori bazlı olarak dinamik oluşturulur.
* **Harici API Entegrasyonu:** Hacker News gibi harici API'ları kolayca entegre etme altyapısı mevcuttur.

| **İkon Seti** | [Bootstrap Icons](https://icons.getbootstrap.com/) | Görsel zenginlik ve kullanıcı deneyimi için. |
| **Markdown İşleme** | [Marked.js](https://marked.js.org/) | Markdown metnini HTML'e çevirmek için. |
| **Kod Vurgulama** | [Highlight.js](https://highlightjs.org/) | Kod bloklarını renklendirmek için. |
| **Formül İşleme** | [KaTeX](https://katex.org/) | Hızlı ve doğru matematiksel formül render etme için. |

├── README.md

├── index.html # Ana sayfa (Blog Feed). 
├── md.html # Dokümantasyon ve Teknik Arşiv sayfası.

├── md.css # Dokümantasyon sayfası için özel stil.

├── /js/ # Tüm JavaScript dosyaları (main.js, md.js, utils.js). 
├── /documents/ # Markdown (.md) doküman dosyalarının konulacağı dizin. 
└── /posts/ # Detaylı makalelerin konulacağı dizin.
